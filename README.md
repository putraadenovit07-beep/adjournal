# AdJournal

Paid Traffic & Adsense Tracker untuk blogger Indonesia.
Dibangun dengan React + Vite, data tersimpan di localStorage.

## Fitur
- Dashboard performa kampanye
- Multi-campaign management
- Catat data harian (Spend, Klik, Tayangan, Penghasilan)
- Auto-konversi USD → IDR (live rate dari API)
- Goals & Milestone tracker
- Analitik ROI & Revenue
